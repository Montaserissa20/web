# Pet Connect Hub

React + Vite project using TypeScript, Tailwind CSS, and shadcn-ui components.

## Getting started

```sh
# Install dependencies
npm install

# Start the dev server
npm run dev

# Lint the project
npm run lint

# Build for production
npm run build

# Preview the production build
npm run preview
```

## Tech stack

- Vite
- React (TypeScript)
- Tailwind CSS
- shadcn-ui
